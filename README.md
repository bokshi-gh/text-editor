# cvim

A terminal-based text editor inspired by Vim.

---

## Features

- Fast and lightweight
- Terminal-based interface
- Vim-like behavior

---

## Installation

- Install via the installer script:

  ```bash
  curl -fsSL https://raw.githubusercontent.com/bokshi-gh/cvim/main/scripts/install.sh | sudo bash
  ```

> Requirements: [cURL](https://curl.se/), [GNU Make](https://www.gnu.org/software/make/), and [GCC](https://gcc.gnu.org/)

> You can check the source code of the installer script [here](https://raw.githubusercontent.com/bokshi-gh/cvim/main/scripts/install.sh).
